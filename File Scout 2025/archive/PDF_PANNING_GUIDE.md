# PDF Panning Guide - Complete Navigation

## 🖱️ **How to Pan in PDF Preview**

File Scout 3.2 now provides **multiple ways to pan** (navigate) around PDF documents for the best viewing experience.

---

## 🎯 **Panning Methods Available**

### **1. Scrollbar Panning (Always Available)**
```
┌─────────────────────────────────────┐
│         ▲ Vertical Scrollbar         │
│                                     │
│ ◀ Horizontal Scrollbar ▶           │
│                                     │
│         ▼ Vertical Scrollbar         │
└─────────────────────────────────────┘
```
- **Vertical scrollbar** - Scroll up/down through pages
- **Horizontal scrollbar** - Scroll left/right for wide content
- **Mouse wheel** - Scroll vertically
- **Shift + Mouse wheel** - Scroll horizontally

### **2. Drag Pan Mode (New Feature)**
```
Toolbar: ✋ Drag Pan  ⟲ Reset View
```
**How to use:**
1. **Click "✋ Drag Pan"** button to enable pan mode
2. **Cursor changes** to open hand (✋)
3. **Click and drag** on the PDF to pan around
4. **Cursor changes** to closed hand (✊) while dragging
5. **Click "✋ Drag Pan"** again to disable pan mode

### **3. Keyboard Navigation**
- **Arrow Keys** - Navigate in all directions
- **Page Up/Down** - Move up/down one screen
- **Home/End** - Jump to top/bottom of document
- **Spacebar** - Page down (when not in search box)

## 🧪 **Step-by-Step Panning Tutorial**

### **Method 1: Scrollbar Panning**
1. **Open any PDF** in File Scout 3.2
2. **Go to PDF Viewer tab**
3. **Use vertical scrollbar** to scroll up/down
4. **Use horizontal scrollbar** to scroll left/right
5. **Use mouse wheel** for quick vertical scrolling

### **Method 2: Drag Pan (Recommended for large documents)**
1. **Open a PDF** and zoom in (🔍+ button)
2. **Click "✋ Drag Pan"** button in toolbar
3. **Notice cursor** changes to open hand (✋)
4. **Click and hold** anywhere on the PDF
5. **Drag mouse** to pan around the document
6. **Release mouse** to stop panning
7. **Click "✋ Drag Pan"** again to disable

### **Method 3: Reset View**
1. **After panning/zooming** around a document
2. **Click "⟲ Reset View"** to return to fit width
3. **Automatically scrolls** to top-left corner
4. **Perfect for** getting oriented after detailed inspection
